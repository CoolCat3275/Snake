# Snake

A snake game for windows made using C++.

# How To Use

Download the ```Snake.exe``` file and run it. That simple.

# What It Does (So Far)

<ul>
<li>Creates Window</li>
<li>Draws Snake/Food</li>
<li>Eat Food and Gain Length</li>
<li>Kills You When Touch Walls Or Yourself</li>
<li>Keeps score</li>
</ul>
